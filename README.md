# Stella Repairer - ドーム視点テスト

Pygameを使用して開発された、全天球（ドーム視点）風の操作を取り入れたシューティング・探索ゲームです。星空と星座を背景に、現れる「バグ」を撃破し、「エネルギーオーブ」を集めながらゲームクリアを目指します。

## 特徴
- **ドーム状視点**: マウスまたはキーボード（WASD）で視点を回転させながら360度の星空を探索できます。
- **難易度選択**: 月の満ち欠け（新月〜二十六夜月）になぞらえた複数の難易度が用意されています。
- **美麗な演出**: 星座の描画、BGM、各種効果音による没入感のあるプレイ体験が可能です。

## 必要要件
- Python 3.x
- Pygame (`pip install pygame`)

## 遊び方

### 起動方法
リポジトリをクローンした後、ディレクトリ内で以下のコマンドを実行してゲームを起動します。

```bash
# 必要に応じて仮想環境を有効化してください
python main.py
```

### 操作方法
**[タイトル画面 / 難易度選択画面]**
- `Space` / `Enter` / 左クリック : ゲームの開始・難易度選択へ
- `A` / `D` または `←` / `→` : 難易度変更
- 右上のボタン (または `H` キー) : 難易度詳細情報の表示/非表示

**[ゲーム中]**
- **視点移動**: 
  - `W`, `A`, `S`, `D`キー: 視点移動
  - `C`キー: マウス視点モードの切り替え（マウスの動きで視点を移動）
- **アクション**:
  - `左クリック`: レティクル中央に向けてショットを発射（バグの撃破）
- **アビリティ**:
  - `E`キー: エネルギーオーブが溜まっている場合、アビリティを発動します。

## ディレクトリ構成
- `main.py` : エントリーポイント
- `src/` : ソースコード群
  - `core/` : カメラシステムやゲームの進行ロジック
  - `entities/` : バグやエネルギーオーブのクラス
  - `managers/` : 各種エンティティ・アビリティの管理
  - `ui/` : ゲーム描画やUIコンポーネント
  - `tools/` : 開発用ユーティリティ
- `source/` : 音声ファイル (BGM / SE) や画像素材
- `data/` : 星座データなどのJSON
